# 未來工程師職業指南

在 Agentic Coding 時代，如何定位、轉型與保持價值。

這是一份關於 AI coding 時代工程師職涯定位的完整主題電子書。它討論的不是「AI 會不會取代工程師」這個粗糙問題，而是：哪些工程工作會被壓縮、哪些能力會升值，以及個人工程師與技術團隊應該如何調整自己的位置。

核心主張：

> 不要把自己訓練成 AI 的打字員；要把自己訓練成 AI 的設計者、約束者、審核者、事故承擔者與系統簡化者。

## 適合誰讀

- Junior / Mid / Senior 工程師：判斷自己目前的職涯風險與下一步能力投資。
- Tech Lead / Architect 候選人：理解 AI-assisted SDLC、review gate、ownership 與 production risk。
- 工程管理者：重新思考團隊在 AI coding 時代的任務分配、Junior 成長路徑與治理流程。
